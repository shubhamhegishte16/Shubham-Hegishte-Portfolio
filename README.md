# 🚀 Shubham's Portfolio

Welcome to my digital playground! This is a high-performance, interactive portfolio showcasing my journey as a **Diploma CS Student & Developer**. Built with a focus on immersive animations, real-time physics, and a premium "hacker-aesthetic" design.

> [!TIP]
> This portfolio is a prototype **vibecoded** by **Antigravity**, **Claude**, and **Gemini** all together! 🚀✨

> [!IMPORTANT]
> This portfolio is best experienced on a modern desktop browser to fully enjoy the physics and 3D effects.

## ✨ Key Features

-   **🕹️ Interactive Physics**: Skill balls that react to your mouse and gravity (powered by `Matter.js`).
-   **🌌 3D Backgrounds**: Immersive starfield and particle effects (powered by `Three.js`).
-   **⚡ Silky Animations**: Smooth scroll triggers and page transitions (powered by `GSAP`).
-   **🖱️ Custom Cursor**: Magnetic, liquid-style cursor that interacts with elements.
-   **📱 Fully Responsive**: Modern layout that scales beautifully across devices.
-   **🖋️ Dynamic Typing**: Animated headlines and subtexts.
